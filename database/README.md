# DevFolio Database Schema

This directory contains the MySQL database setup scripts for the DevFolio application.

## Setup Instructions

1. Install MySQL Server if not already installed
2. Open MySQL Workbench or MySQL Command Line
3. Run the `schema.sql` file to create all tables
4. Configure database credentials in the backend `.env` file

## Database Structure

- **users** - User accounts and profiles
- **projects** - Portfolio projects
- **posts** - Blog articles
- **awards** - Achievements and badges
